# INOWYA - Location Analyzer

Privacy-first, open-source tool for analyzing location history.

Supports:
- GPX / KML / GeoJSON
- Google Takeout (Location History + Semantic Locations)
- ZIP archives
- CSV exports

Features:
- Stop & trip detection (background worker – no freeze)
- Speed-based coloring
- Heatmap
- Date filtering + single-day timeline scrubber
- Dark/light theme
- 100% offline – runs completely in your browser

Just open `INOWYA.html` and drop your files.

**No data ever leaves your computer.**
